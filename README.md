CALCULADORA DE COSTO DEL PRODUCTO

Este proyecto consiste en un programa desarrollado en Python que automatiza el proceso de registro de un producto, validando la entrada de datos (nombre, precio y cantidad) y calculando el costo total.

 DESCRIPCION DEL PROYECTO
 
El programa guía al usuario a través de una serie de pasos para ingresar la información de un producto. Incluye mecanismos de control de errores para asegurar que los datos ingresados sean lógicos y del tipo correcto, evitando que la aplicación se cierre por entradas inválidas.

CARACTERISTICAS PRINCIALES

VALIDACIÓN DE TEXTO: Asegura que el nombre del producto solo contenga letras.
VALIDACIÓN NÚMERICA: Verifica que el precio y la cantidad sean números positivos.
MANEJO DE EXCEPCIONES: Utiliza bloques try-except para gestionar errores de tipo de dato.
CALCULO AUTOMATICO: Genera el total multiplicando el precio por la cantidad.

 TECNOLOGIAS UTILIZADAS
 
LENGUAJE: Python 3.x
LÓGICA: Estructuras de control (while, if-else) y manejo de excepciones.
 Lógica del Programa (Diagrama de Flujo)
 
EL FLUJO DEL TRABAJO SIGUE LA SIGUIENTE ESTRUCTURA LOGICA:

INICIO: Arranque del sistema.
ENTRADA DE NOMBRE: Lectura y validación de caracteres alfabéticos.
ENTRADA DE PRECIO: Validación de número flotante positivo.
ENTRADA DE CANTIDAD: Validación de número entero positivo.
PROCESAMIENTO: Cálculo del costo_total = precio * cantidad.
SALIDA: Impresión de un resumen detallado con los resultados.
